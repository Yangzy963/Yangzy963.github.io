## Markdown语法指南
   Markdown 是一种轻量级标记语言，它通过简单的符号实现排版，让作者专注于内容而非格式。本文将为你系统性地梳理常用的 Markdown 写作技巧。
## 一、 标题与文本样式  
1. 多级标题
通过在行首添加 # 来定义标题级别，最多支持六级
```
# 一级标题
## 二级标题
...
###### 六级标题
```
2. 文本强调  
加粗： 使用 **文本**  
斜体： 使用 *文本*   
<u>下划线</u>：使用 HTML 标签 <u>文本</u>
3. 上下标与表情  
下标：使用 H~2~O 或 HTML 标签 H<sub>2</sub>O  
表情符号：支持标准的 Emoji 代码，如 :smile: 😄  
## 二、 列表与任务管理  
1. 有序与无序列表  
    - 有序列表：使用 1. 、2.   。例如：
       1. 打开冰箱  
	   2. 把大象塞进去  
    - 无序列表：使用 - 或 * .例如： 
	  * 项目A  
	  - 项目B  
2. 任务列表  
用于管理待办事项:  
- [x]已完成事项  -->    ``` - [x]事项内容```  
- []未完成事项   -->    ``` - []事项内容```  
## 三、引用、代码与链接  
1. 引用  
在段落开头使用 > :  
>这是一段引用文字  
2. 代码展示  
使用反引号标注行内代码 printf()，或者使用三个反引号包裹代码块：  
```c 
int main()
{
	return 0;
}
```
3. 链接与参考  
普通链接：[百度](https://baidu.com"百度网站")  ```实现方式：[百度](https://baidu.com"百度网站")```  
引用链接：[百度][id] ```在文中写 [百度][id]，在文末定义  [id]: https://baidu.com```  
自动链接：直接输入网址即可自动转换，如 http://yangzongyi.cn    
## 四、结构化数据：表格与脚注  
1. 表格排版  
使用|分隔单元，使用 - 分隔表头。通过 : 控制对齐方式：  
 :- -  左对齐    
  -- -：  右对齐  
 :- -: 居中对齐  
|姓名|年龄|性别|
|:--|--:|:--:|
|杜夫|58|男|
|李晴兆|71|女|
2. 脚注  
在需要解释的词语后使用 [^序号]  , 并在文末进行说明：[^序号]:脚注的内容。若被解释的内容与脚注同一行则不会生效。    示例:未雨绸缪[^解释]     
## 五、其他元素  
- 分割线：使用三个减号---即可生成横线。  
- 数学公式:Markdown支持嵌入LaTex格式的数学公式，用于科学记录和学术写作。   
  
[id]: https://baidu.com   
 [^解释]:《诗经·豳风·鸱鸮》：「迨天之未阴雨，彻彼桑土，绸缪牖户。」（绸缪：修补）意思是鸱鸮在未下雨时就啄来桑树皮，修补鸟巢。后用「未雨绸缪」比喻事先作好准备。  
[^伦理]:指人与人相处、社会行为中应遵循的道德准则、规范及哲学思考。
