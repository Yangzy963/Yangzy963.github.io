# 羊了个羊技能

## 概述

此技能用于处理与"羊了个羊"小游戏相关的任务。

## 游戏规则 (核心逻辑)

在执行游戏相关任务时，必须严格遵守以下规则：

1.  **游戏目标**：玩家需要通过消除卡片来完成关卡，最终目标是清空所有的卡片。
2.  **消除规则**：
    * 游戏界面中会出现多个图案，玩家需要点击图案将其放入底部的槽中。
    * 当槽中有**三个相同**的图案时，这三个图案会自动消除。
    * 玩家需要尽量避免槽中积累过多不同的图案，以免无法继续消除。
    * **硬性约束**：严禁点击收集槽里的图案；**严禁点击屏幕底部蓝色区域的三个道具按钮（移出、撤回、洗牌）**；严禁观看广告。
    * **强制规避坐标**：在游戏主界面中，禁止点击 Y 坐标大于 900 的区域（道具栏所在位置）。
    * **流程控制**：请持续推进游戏进程，游戏通关后继续下一关，游戏失败后重新开始。
    * 卡片是方形的，并且亮色的卡片是可以点击的，暗色的卡片是被亮色的卡片覆盖在下面的，需要将亮色的卡片点开才能点击暗色的卡片。
    * 点击卡片就能将卡片放入卡槽，如果卡槽中有3个相连并且相同的卡片就能将这3个相同的卡片抵消，从而腾出3个新的位置。
    * 卡槽最大容量是7个，满了7个就会导致游戏失败（指的是这7个卡片中没有三个相同的卡片的情况，比如2个萝卜2个玉米2个草1个萝卜这种情况会导致游戏失败，不是说7个卡槽中有三个萝卜就能自动抵消，还有一个要求是三个萝卜必须连在一起，2个草2个玉米2个萝卜1个萝卜这种情况就行）。
3.  **游戏界面**：图案通常以堆叠的方式呈现，玩家需要逐层消除。
4.  **关卡设计**：游戏包含多个关卡，随着关卡的推进，图案的复杂度和数量会增加。
5.  **策略性**：玩家需要规划消除顺序，以避免槽中积累过多无法消除的图案。

## 使用场景

- 用户要求"打开抖音搜索羊了个羊"
- 用户要求"玩羊了个羊"
- 用户要求"搜索羊了个羊并完成一局游戏"

## 工作流程

1.  **识别任务**：当用户任务涉及羊了个羊游戏时触发。
2.  **执行辅助**：根据上述游戏规则，调用脚本或指令提供辅助操作。
3.  **返回结果**：将结果返回给 Agent 继续执行手机操作。

## 注意事项

- 实际的游戏操作仍然通过手机控制完成。
- 此技能主要用于识别和辅助处理游戏相关任务。
- **弹窗处理**：若误触导致弹出“移出道具”、“撤回道具”或“洗牌道具”的对话框（显示有“获得”和“不，谢谢”选项），**必须立即点击“不，谢谢”或右上角的关闭按钮 [X]**，严禁点击带有视频图标的“获得”按钮。
- **操作原则**：在无道具、无广告的前提下，通过纯策略进行游戏推进。
