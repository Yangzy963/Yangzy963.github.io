个人博客创建流程

Thursday, January 29, 2026
1 minute read

一、安装流程

  1. 下载hugo压缩包
  2. 在含有hugo.exe的文件夹下输入cmd
  3. 执行hugo new site my_blog :创建文件夹,以my_blog为例 系统提示按照图中的几个步骤就能搭建个人博客 [图片]
  4. 将hugo.exe配置系统变量，关掉目前所有打开的 cmd / PowerShell 窗口，重新开放一个才生效
  5. 切换到刚才创建的文件目录： cd my_blog
  6. 输入：hugo server –buildDrafts 出现如下页面说明服务已经搭好，还没安装主题 [图片]
  7. 前往https://themes.gohugo.io/下载主题压缩包，存放到所创建文件夹下的themes中解压缩。 此时需要避免出现 my_blog\themes\stack\stack的情况，此时 需要将第二个stack中的所有内容剪切到第一个stack中，并删除第二个stack文件夹。{因为 Hugo 在第一层 themes/stack/ 下找不到 layouts 和 theme.toml，它就不知道该如何渲染页面，所以直接给你显示了 404 错误。}
  8. 在…\my_blog>路径下输入：hugo new post/Article01.md 快速创建一个新的 Markdown 内容文件 注意上述中的post如果改为posts可能会导致Article01.md中的内容无法正常显示
  9. 在创建的my_blog文件下中在文本编辑器中打开 hugo.toml ，在末尾添加：theme = ‘stack’，注意此处的主题名要与theme文件夹下的相同。
  10. 进入content\posts中找到Article01.md,用文本编辑器打开，将draft = true中的true改为false，让这篇文章从“草稿”变成“正式内容”。
  11. 输入：hugo server –buildDrafts 或 hugo server 出现如下页面说明服务已经搭好 [图片]
